# TabVault Pro Heavy User Interview Plan

目标：找 20 个重度浏览器用户，只验证“他们会在什么场景下保存”以及“他们会不会回来恢复”。

## 访谈对象

- 开发者
- 研究型用户
- 内容作者
- 销售 / 招聘 / 投资从业者

## 开场问题

- 你平时什么时候会同时打开很多网页？
- 你关闭浏览器或切换任务前，最怕丢掉的是什么？

## 核心问题

- 你会在什么场景下愿意点击一次“保存会话”？
- 你之后通常是怎么重新回到这批页面的？
- 如果这个工具能帮你直接恢复，你会不会真的回来用？
- 备注、收藏、最近恢复里哪个更能帮助你找回工作上下文？
- 你对“手动同步”最大的顾虑是什么？

## 观察点

- 是否能自然理解“会话”而不是“标签组”
- 是否知道恢复默认在新窗口打开
- 是否在意 pinned 标签页状态
- 是否愿意为跨设备连续性和误删恢复付费

## 结束记录

- 受访者画像
- 最强触发场景
- 是否出现真实恢复需求
- 对付费价值的原话

OneTab Plus 搜索功能测试指南

注意：本测试需要在Chrome浏览器中安装OneTab Plus插件后进行。
测试1：单栏模式下的搜索
验证在单栏显示模式下，搜索结果是否正确显示为单栏格式
测试步骤：
  1. 打开OneTab Plus插件弹窗
  2. 确保当前显示模式为单栏（点击布局切换按钮直到显示单栏图标）
  3. 在搜索框中输入关键词（如"test"、"google"等）
  4. 观察搜索结果的显示格式
预期结果：搜索结果应该以单栏格式显示，每个匹配的标签页垂直排列
测试2：双栏模式下的搜索
验证在双栏显示模式下，搜索结果是否强制显示为单栏格式
测试步骤：
  1. 打开OneTab Plus插件弹窗
  2. 切换到双栏显示模式（点击布局切换按钮直到显示双栏图标）
  3. 确认标签组以双栏格式显示
  4. 在搜索框中输入关键词
  5. 观察搜索结果的显示格式
  6. 清空搜索框，观察是否恢复双栏显示
预期结果：
  • 搜索时：搜索结果强制显示为单栏格式
  • 清空搜索后：恢复双栏显示模式
测试3：三栏模式下的搜索
验证在三栏显示模式下，搜索结果是否强制显示为单栏格式
测试步骤：
  1. 打开OneTab Plus插件弹窗
  2. 切换到三栏显示模式（点击布局切换按钮直到显示三栏图标）
  3. 确认标签组以三栏格式显示
  4. 在搜索框中输入关键词
  5. 观察搜索结果的显示格式
  6. 清空搜索框，观察是否恢复三栏显示
预期结果：
  • 搜索时：搜索结果强制显示为单栏格式
  • 清空搜索后：恢复三栏显示模式
测试4：搜索功能验证
验证搜索功能的基本操作是否正常
测试步骤：
  1. 在搜索框中输入存在的标签页标题关键词
  2. 验证匹配的标签页是否正确显示
  3. 点击搜索结果中的标签页链接
  4. 验证标签页是否正确打开
  5. 测试搜索结果中的删除按钮
  6. 测试"恢复全部"按钮
预期结果：
  • 搜索结果准确匹配
  • 点击链接正确打开标签页
  • 删除和恢复功能正常工作
  • 所有操作都在单栏格式下进行
修改总结
本次修改的主要内容
修改内容：
  • 移除了SearchResultList组件对废弃的useDoubleColumnLayout字段的依赖
  • 强制搜索结果以单栏格式显示，不受用户当前布局模式影响
  • 保持非搜索状态下的正常多栏显示功能
  • 确保搜索功能的所有操作（打开、删除、恢复）正常工作
测试完成后：如果所有测试都通过，说明搜索结果强制单栏显示功能已成功实现。

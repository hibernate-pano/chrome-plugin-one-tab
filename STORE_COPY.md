# TabVault Pro Store Copy

## One-liner

Save browser work sessions, find them later, and restore them when you need to continue.

## Short Description

把当前窗口保存成可找回、可恢复的工作会话。

## Long Description

TabVault Pro 帮你把浏览器里的工作现场保存下来，而不是只做一个普通的标签收纳盒。

适合开发者、研究者、内容作者、销售、招聘和任何长期同时打开大量网页的人。你可以在中断前保存当前窗口，之后按会话名称、备注、标题或 URL 快速找回，并在新窗口中恢复整组工作内容。

### 你可以做什么

- 保存当前窗口为工作会话
- 搜索会话名称、备注、标签标题和 URL
- 在新窗口中恢复整个会话
- 收藏重要会话并补充备注
- 从最近保存或最近恢复入口快速回到工作现场
- 保留或忽略固定标签页
- 导入 / 导出 OneTab 数据
- 登录后手动上传到云端或从云端下载

### 当前同步说明

当前版本提供手动同步，不承诺实时自动同步。

- 上传：把本地会话推送到云端
- 下载：把云端数据拉回本地
- 每次都会明确说明是“覆盖”还是“合并”

## FAQ Draft

### 这是不是 OneTab 替代品？

它支持 OneTab 导入，但定位不是简单替代品，而是“工作会话保险箱”。

### 现在是自动同步吗？

不是。当前版本是手动同步。

### 恢复会话会覆盖我当前窗口吗？

不会。默认会在新窗口中恢复整组会话。
